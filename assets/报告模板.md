## Callout 格式说明

| 格式 | 用途 | 颜色 |
|------|------|------|
| `> [!NOTE]` | 基本信息 | 蓝色 |
| `> [!TIP]` | 提示建议 | 绿色 |
| `> [!WARNING]` | 风险警告 | 橙色 |
| `> [!SUCCESS]` | 正面信号 | 绿色 |
| `> [!INFO]` | 一般信息 | 蓝色 |
| `> [!CAUTION]` | 风险提示 | 红色 |
| `> [!SUMMARY]` | 总结 | 紫色 |

---

| 变量 | 说明 | 示例 |

| `{{股票代码}}` | 6位数字代码 | 600684 |

| `{{X}}` | 百分比 | 5 |

## 使用方法

1. 复制模板内容
2. 替换所有 `{{变量}}` 为实际数据
3. 根据分析结果填充各专家观点
4. 保留 Callout 格式以保持 Obsidian 兼容
5. 保存到 `stock-reports/` 目录，文件名格式：`{{股票代码}}_{{股票名称}}_{{YYYYMMDD}}.md`
