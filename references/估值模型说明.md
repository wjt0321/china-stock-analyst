# A股常用估值模型说明

> 本文档供基本面大师和量化模型师参考使用

## 一、市盈率（P/E）

### 1.1 静态市盈率
```
静态P/E = 当前股价 / 上年度每股收益(EPS)
```
- 优点：数据客观，不受季节影响
- 缺点：滞后性强，无法反映最新变化

### 1.2 动态市盈率
```
动态P/E = 当前股价 / 预期本年度每股收益
```
- 优点：反映市场预期
- 缺点：预测可能偏差较大

### 1.3 市盈率估值参考区间

| 行业 | 低估 | 合理 | 高估 |
|------|------|------|------|
| 银行 | 5-8 | 8-12 | >12 |
| 地产 | 4-7 | 7-15 | >15 |
| 消费 | 15-20 | 20-30 | >30 |
| 科技 | 25-35 | 35-50 | >50 |
| 医药 | 20-30 | 30-45 | >45 |

### 1.4 使用注意事项
- 适用于盈利稳定的企业
- 周期性行业需用周期调整P/E
- 亏损企业不适用

---

## 二、市净率（P/B）

### 2.1 计算公式
```
P/B = 当前股价 / 每股净资产(BV)
```

### 2.2 估值参考

| 区间 | 含义 |
|------|------|
| <1 | 破净，可能被低估 |
| 1-3 | 正常区间 |
| >3 | 可能被高估 |

### 2.3 适用场景
- 银行、保险等重资产行业
- 困境反转企业
- 资产清算价值评估

---

## 三、DCF（现金流折现模型）

### 3.1 基本公式
```
企业价值 = Σ (FCF_t / (1+WACC)^t)
```
其中：
- FCF_t = 第t年自由现金流
- WACC = 加权平均资本成本

### 3.2 自由现金流计算
```
FCF = 净利润 + 折旧摊销 - 资本支出 - 营运资本增加
```

### 3.3 DCF估值步骤

1. **预测期（5-10年）**
   - 分析历史营收增长
   - 假设未来增长率
   - 预测利润率变化

2. **永续增长期**
   - 终值 = FCF_n × (1+g) / (WACC-g)
   - 通常取 g = 2-3%

3. **折现计算**
   - 将未来现金流折现到现值
   - 加总得到企业价值

### 3.4 关键参数

| 参数 | 参考范围 |
|------|----------|
| WACC | 8-12%（视风险调整） |
| 永续增长率 | 2-3% |
| 增长期 | 5-10年 |

---

## 四、PEG（市盈率相对盈利增长）

### 4.1 计算公式
```
PEG = P/E / 净利润增长率(G)
```

### 4.2 估值判断

| PEG值 | 判断 |
|-------|------|
| <0.5 | 严重低估 |
| 0.5-1 | 被低估 |
| 1-1.5 | 合理 |
| 1.5-2 | 轻微高估 |
| >2 | 严重高估 |

### 4.3 适用条件
- 适用于高成长企业
- 增长率需 > 15% 才可靠
- 成熟企业不适用

---

## 五、股息率模型

### 5.1 计算公式
```
股息率 = 每股股息 / 股价
```

### 5.2 估值标准
- 高股息率 > 4%
- 适中 2-4%
- 低股息率 < 2%

### 5.3 适用企业
- 成熟稳定企业
- 银行、公用事业
- 高分红传统行业

---

## 六、行业特有问题

### 6.1 房地产
- 常用 NAV（净资产价值法）
- 需考虑土地储备、在建项目
- 负债率是关键风险指标

### 6.2 银行
- 用 P/B 而非 P/E
- 关注不良贷款率
- 拨备覆盖率

### 6.3 科技/互联网
- 用户数、MAU 估值
- PS（市销率）更适用
- 关注用户获取成本

---

## 七、估值风险提示

> [!WARNING]
> 1. 单一估值方法有局限，建议多种方法交叉验证
> 2. 预测假设可能不成立
> 3. 市场情绪可能导致长期偏离价值
> 4. 宏观环境变化影响所有估值假设
